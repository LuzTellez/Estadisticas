# Estadisticas

#### Create virtual enviroment
```
py -m venv env
```

#### Activate virtual enviroment
```
env\Scripts\activate
```

#### Instaling the package of the enviroment
```
pip install -r requirements.txt
```

#### Deactivating virtual enviroment
```
env\Scripts\deactivate.bat
```
